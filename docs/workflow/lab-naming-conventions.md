# Lab Naming Conventions

Use deterministic, sortable names.

## Format

`lab-XX-topic-goal`

Examples:
- `lab-00-cycle-sim`
- `lab-01-led-blink-asm`
- `lab-02-timer0-ctc`
- `lab-03-coop-scheduler`

## Layer Folder Format

`L<layer-number>_<layer-name>`

Examples:
- `L1_register-control`
- `L2_timing-mastery`
- `L3_concurrency`
