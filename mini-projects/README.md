# Mini Projects

Mini projects turn lab primitives into integrated systems.

## Difficulty Tiers

### Beginner

- `mp-01-deterministic-led-engine`
  - Multi-pattern LED engine with timer-based updates.
  - No blocking delays.

### Intermediate

- `mp-02-motor-pwm-controller`
  - PWM motor control with bounded update loop.
  - Input filtering and safe state fallback.

### Advanced

- `mp-03-soft-real-time-task-executive`
  - Multi-task cooperative executive.
  - Telemetry on jitter, misses, and utilization.
